# Savia Stellar Bloom 54

Stellar blockchain-powered savings and investment platform.

## Stack

- **Vite + React + TypeScript** — Frontend
- **Stellar SDK** — Blockchain integration (@stellar/freighter-api, @stellar/stellar-sdk)
- **shadcn/ui** — UI component library with Tailwind CSS

## Quick Start

```bash
npm install
npm run dev
```

Requires Freighter wallet browser extension.
